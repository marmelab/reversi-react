# reversi-react

Experimental react isomorphic reversi game for browser

Here is the associated blog post [React Isomorphique en Pratique](https://marmelab.com/blog/2016/12/21/react-isomorphique-en-pratique.html)

## Install

```
make install
```

## Run

```
make run
```

## Test

```
make test
```

## Lint

```
make lint
```
